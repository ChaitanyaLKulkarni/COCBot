# COC Bot
Live feed of clash of code stats for twitch streamers!

# Running locally
## Running the entire system

- Populate an `.env` file in root. (use `.env.sample` for help.)
- taskfile: `task`
- or, docker-compose: `docker-compose up --build`
- or, foreman: `foreman start`
